基于 Python（Pandas/Numpy）的数据分析与业务自动化项目集合
该项目集合聚焦于Excel 数据处理、业务规则自动化执行与数据校验，利用 Python 的 Pandas 和 Numpy 库，实现多个业务场景下的自动化流程，包括奖励发放、体测成绩计算、客户分配等，显著提升数据处理效率与准确性。
项目功能模块
1. 京东卡奖励自动化发放模块
读取 “京东卡” 和 “用户应得奖励金” 两类 Excel 数据，按用户奖励金额自动匹配京东卡面值并分配。
支持多保单号合并展示与按用户 ID 拆分输出结果文件，同时通过金额求和校验数据一致性，确保奖励发放准确无误。
2. 体测成绩自动评分模块
处理 “18 级高一体测成绩汇总” 和 “体测成绩评分表” 数据，完成数据清洗（如字符串转数值、缺失值处理）。
依据体测项目（男 1000 米、男 50 米跑、男跳远等）的评分规则，自动计算每个项目的得分，实现体测成绩的批量标准化评分。
3. 客户分级分配模块
读取 “坐席保留明细” 和 “客户明细” 数据，按客户 “在投等级”（如 A、B、C 类）与坐席承接能力（如 70W+、10-70W 等区间），自动化分配客户资源。
包含数据打乱随机分配逻辑，以及分配结果的多维度校验（如分配数量一致性、坐席承接量匹配度等），保障客户分配的公平性与准确性。
技术栈
核心库：Pandas（数据读取、清洗、分组、合并）、Numpy（数值计算支持）。
业务场景：Excel 数据交互、自定义函数实现业务规则（如金额匹配、成绩映射、客户分配逻辑）、多步骤数据校验保障结果准确。
应用价值
效率提升：替代人工 Excel 操作，批量处理多维度数据，大幅减少重复劳动。
准确性保障：通过自动化逻辑和数据校验环节，避免人工计算 / 分配的失误。
场景复用：各模块的业务逻辑可根据相似场景（如其他奖励发放、评分标准、资源分配）进行适配，具备一定通用性。
